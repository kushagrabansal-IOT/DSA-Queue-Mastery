## 🎯 Learning Objectives
- Implement Circular Queue from scratch (array-based)
- Master Deque for sliding window problems
- Use Priority Queue (heapq) for K-way merges
- Apply BFS with queue for shortest path problems
- Understand queue vs deque vs priority queue tradeoffs

## 📋 Problem Statements
### P1 — Sliding Window Maximum: `[1,3,-1,-3,5,3,6,7],k=3→[3,3,5,5,6,7]`
### P2 — BFS Shortest Path: Min steps in grid from (0,0) to (n-1,n-1)
### P3 — Top K Frequent: `[1,1,1,2,2,3],k=2→[1,2]`
### P4 — Merge K Sorted Lists: Merge k sorted arrays efficiently
### P5 — Task Scheduler: `["A","A","A","B","B","B"],n=2→8`
